## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) `v18+`
- A [Supabase](https://supabase.com/) project *(free tier works!)*

### ⚙️ Installation

```bash
# 1. Clone the repository
git clone https://github.com/Gone001/Get-quick-job.git
cd Get-quick-job

# 2. Install dependencies
npm install

# 3. Add your environment variables
# Create a .env.local file and add your Supabase project credentials

# 4. Start development server
npm run dev
```

🎉 Open [http://localhost:3000](http://localhost:3000) in your browser!

> 💡 Supabase credentials can be found in your Supabase Dashboard → **Settings → API**

## 🌐 Pages & Routes

| Route | Page | Access |
|:------|:-----|:-------|
| `/` | 🏠 Landing Page | Public |
| `/auth/login` | 🔐 Login / Sign Up | Public |
| `/dashboard/worker` | 👷 Browse & Apply to Jobs | Auth Required |
| `/dashboard/recruiter` | 🏢 Post Jobs & Find Workers | Auth Required |
| `/dashboard/profile` | 👤 Profile & Application History | Auth Required |
| `/jobs/post` | 📝 Post a New Job | Auth Required |

